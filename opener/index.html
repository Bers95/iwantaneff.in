This is for opening cyberlocker links. For example, groups of links like the following

http://rapidshare.com/files/12345
http://rapidshare.com/files/12345
http://rapidshare.com/files/12345
http://rapidshare.com/files/12345
http://rapidshare.com/files/12345
http://rapidshare.com/files/12345

You also have the option to parse links (anything with 'http://' will be parsed). When opening links, if you have direct-downloads turned on, the tabs will close automatically.
