About StatusVia

In laymen's terms, StatusVia emulates Facebook applications and allows you to post status updates as if you were using that application.

Inspired by a reddit post, StatusVia is a simple online tool that lets you post Facebook status updates that appear to come from funny and unique applications. For example, when someone posts a status update using the Facebook application on their iPhone, the Facebook website will say "a few seconds ago via iPhone" under the status. StatusVia simulates different applications, both official applications and joke applications, so you can post status updates that instead say "a few seconds ago via Fedex" or "a few seconds ago via Smoke Signals".

StatusVia is not affiliated or associated with Facebook in any way, and does not collect any information from you other than your IP address.
