Effin Toolset

A suite of tools for web developers


I want an effin toolset is a suite of tools & resources for web developers. These are tools, the web developer David Higgins uses frequently. I want a site where I have instant (worldwide) access to the tools I love the most. Be sure to check out Other Projects for a bigger, much richer series of tools. And if you want to know more about these tools, watch the screencast that explains them in further detail.

 

Break it down, now. Why do I need these tools?


You can use these tools for building websites, and web-apps. If you're annoyed certain tools didn't make the list, fork the project and add your own. Some of the 'tools' are stricly for phun purposes only. Particularly this, this, and this. This domain also serves as a personal playground where I can showcase projects I have been working on, to others. Namely this, this, and this.

You have HTML code you want to place in a <PRE> tag element. You want it entified, so it plays nice with your <PRE> tag. This does that.

iwantaneff.in/textarea/

Sometimes you just need a textarea, right? Sometimes I work on locked down machines, where tools like Textmate, or if on Windows, notepad; aren't available. Also, because Firefox has the spellchecker turned on by default, I can check my spelling really quickly.

iwantaneff.in/facebook.ping

You can use this PNG if you are tired of creating custom graphics for Facebook Open Graph HTML <meta> tag directives. It was renamed PNG to PING because I don't allow hotlinking to this site. This has also been featured in H5BP Redux. Here's a HTML code example:
