Cloudapp roulette is a fun game to play when you're bored. Typically a Cloudapp URL looks like the following: 

http://cl.ly/FBUc

What this program does is generate a random 4 character alphanumberic file ID, and appends it to the cloudapp URL (cl.ly). It then opens fifteen browser tabs / windows so you can browse what Cloudapp item exists there. You can even scour for actual words contained in the Cloudapp ID for extra lulz: 

http://cl.ly/Crap

If nothing is found in a tab, you have lost the game. The aim of the game is to find items appearing in the random tabs that you have opened.

You never know what lies inside a randomly generated file-ID - that's what makes the game so fun. This app allows you to break open the geek zeitgeist, and explore what people are uploading to their Cloudapp accounts. Why? For phun and profit of course.
