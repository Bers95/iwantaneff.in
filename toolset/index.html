iwantaneff.in / toolset

I want an effin toolset is a suite of tools & resources for web developers.
These are tools, the web developer @_higg_ uses frequently.
I want a site where I have instant (worldwide) access to the tools I love the most.
I got the idea from @mathias. He has some tools, too, at: http://mothereff.in/

Devtools

iwantaneff.in/hotpink/

Hotpink. My favourite Dribbble colour. Often-times I would find myself hunting down the hex-code of this colour fervently when I wanted to bang out a website really quickly. Now I have almost 'command line' access to the hex-code via the address bar.

Yes, Please
iwantaneff.in/entifier/

You have HTML code you want to place in a <PRE> tag element. You want it entified, so it plays nice with your <PRE> tag. This does that.

Yes, Please
iwantaneff.in/datauri/

A homage to the wonderful tool Bocoup have released. You need to covert raw bytecode to base64 encoded URI data. This does that

Yes, Please
iwantaneff.in/packer/

You have JavaScript. You need to pack it to a really small size. This does that. Based on Ben Alman's (@cowboy's) packer script. Only works with single line JavaScript. Use this tool to get a single line of JS.

Yes, Please
iwantaneff.in/favicon/

Sometimes Apache servers cry when you don't include a Favicon reference in your HTML file. It is best you contain the data URI snippet with an actual favicon placed in the root of your domain, plus one in the root of the subfolder (if there is one). E.g: http://domain.com/favicon.ico , http://domain.com/folder/favicon.ico

Yes, Please
iwantaneff.in/textarea/

Sometimes you just need a textarea, right? Sometimes I work on locked down machines, where tools like Textmate, or if on Windows, notepad; aren't available. Also, because Firefox has the spellchecker turned on by default, I can check my spelling really quickly.

Yes, Please
iwantaneff.in/beautifier/

Formats and prettifies Ugly JavaScript. Based off the tool here. If, for whatever reason, the tool gets taken offline, here is a backup ;). Plus I never used the sidebar on the official one, so I removed it.

Yes, Please
iwantaneff.in/bookmarklet/

You have JavaScript. You need a bookmarklet. This does that.

Yes, Please
iwantaneff.in/notice/

Your site breaks, fails, and generally just doesn't work in IE6? Re-direct your users to this tongue in cheek notifier page. Yeah, I'm a modern browser fanboy. If this offends you, don't complain. IE6 is officialy dead, anyways. And the latest IE is apparently lightyears ahead of previous versions. If you dislike IE6, you might want to check this: http://isharefil.es/BegE

Yes, Please
iwantaneff.in/slug/

You have a string like this: This is my wondérful élongated blog post tïtle. You want to see a URI/URL friendly version of it that agrees with search engines + browsers, and turns it into something like this: wondrful-longated-blog-post-ttle. This tool will do that.

Yes, Please
iwantaneff.in/JJ Encoder/

The wonderful tool at UTF8.jp reincarnated and open sourced here, on iwantaneff.in. You have JS you need obfuscated beyond recognition. This does that.

Resources

iwantaneff.in/custom.webkit.scrollbar.css

You can link directly to this CSS file, as it's very small in filesize. This has also been featured in H5BP Redux. It's there if you want your site to have custom webkit scrollbars in Chrome/Safari.

Yes, Please
iwantaneff.in/facebook.ping

You can use this PNG if you are tired of creating custom graphics for Facebook Open Graph HTML <meta> tag directives. It was renamed PNG to PING because I don't allow hotlinking to this site. This has also been featured in H5BP Redux. Here's a HTML code example:

<meta property="og:image" content="http://iwantaneff.in/facebook.ping" />

Yes, Please
iwantaneff.in/cursor.css

Did you know CSS has a vast number of options regarding cursors you can use on your HTML5 elements? As ever, you can always hotlink to our cursor.css file which is hosted by Cloudflare's CDN (Content Delivery Network). You might want to visit this page for live demos of all the cursors you can use. Example:

crosshair
move

Misc tools

iwantaneff.in/game-of-cloudapp-roulette/

What this program does is generate a random 4 letter alphanumberic file ID, and appends it to a cloudapp URL. It then opens ten browser tabs / windows so you can see what Cloudapp item exists there.

iwantaneff.in/repeat/

I always hated typing out a FOR loop really quickly when I wanted to repeat text multiple times. Why would someone want to repeat text multiple times? See this: http://isharefil.es/DV7b

Yes, Please
iwantaneff.in/glitcher

You have text. You need to fuck it up, glitch art style. This does that. Based off the tool here.

Yes, Please
iwantaneff.in/shuffler

You have text you don't want to repeat multiple times over, so you shuffle it to make it look unique. Try it, it makes the internet a less samey-place.

Yes, Please
iwantaneff.in/link.list.opener

You have a bunch of cyberlocker links. You need to open them all at once. This does that.
