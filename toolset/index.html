iwantaneff.in/packer/

You have JavaScript. You need to pack it to a really small size. This does that. Based on Ben Alman's (@cowboy's) packer script. Only works with single line JavaScript. Use this tool to get a single line of JS.

iwantaneff.in/favicon/

Sometimes Apache servers cry when you don't include a Favicon reference in your HTML file. It is best you contain the data URI snippet with an actual favicon placed in the root of your domain, plus one in the root of the subfolder (if there is one). E.g: http://domain.com/favicon.ico , http://domain.com/folder/favicon.ico

Yes, Please

Sometimes you just need a textarea, right? Sometimes I work on locked down machines, where tools like Textmate, or if on Windows, notepad; aren't available. Also, because Firefox has the spellchecker turned on by default, I can check my spelling really quickly.

Resources

iwantaneff.in/custom.webkit.scrollbar.css

You can link directly to this CSS file, as it's very small in filesize. This has also been featured in H5BP Redux. It's there if you want your site to have custom webkit scrollbars in Chrome/Safari.

Yes, Please
iwantaneff.in/facebook.ping

You can use this PNG if you are tired of creating custom graphics for Facebook Open Graph HTML <meta> tag directives. It was renamed PNG to PING because I don't allow hotlinking to this site. This has also been featured in H5BP Redux. Here's a HTML code example:

<meta property="og:image" content="http://iwantaneff.in/facebook.ping" />
