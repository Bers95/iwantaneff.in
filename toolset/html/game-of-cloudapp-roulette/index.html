Cloudapp roulette is a fun game to play when you're bored. Typically a Cloudapp URL looks like the following: 

http://cl.ly/FBUc

What this program does is generate a random 4 character alphanumberic file ID, and appends it to the cloudapp URL (cl.ly). It then opens ten browser tabs / windows so you can browse what Cloudapp item exists there. You can even scour for actual words contained in the Cloudapp ID for extra lulz: 

http://cl.ly/Crap

You never know what lies inside a randomly generated file-ID - that's what makes the game so fun. This app allows you to break open the geek zeitgeist!
